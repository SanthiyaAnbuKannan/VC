Computer System Image Map

Click on any component to see its description.

CPU (Central Processing Unit)

The CPU is the brain of the computer. It processes instructions and performs calculations.

RAM (Random Access Memory)

RAM temporarily stores data and programs that the CPU needs while the computer is running.

GPU (Graphics Processing Unit)

The GPU handles graphics rendering and improves visual performance for games and videos.

Storage (Hard Disk / SSD)

Storage devices permanently store the operating system, software, and user data.
